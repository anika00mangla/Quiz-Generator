# Interactive Quiz Generator 🎓

## Overview
This project is a simple tool that turns plain study notes into practice quizzes.  
It’s designed for learners who want a quick way to test themselves and reinforce what they’ve read.

## Features
- Creates multiple‑choice questions automatically from text files
- Runs in the terminal with an interactive interface
- Tracks your score and gives instant feedback
- Easy to extend with your own study material
